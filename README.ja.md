# Kaku

超高速なMarkdownメモアプリ。ホットキーを押して、書いて、閉じる — メモは自動で保存されます。

## 特徴

- **瞬時に起動** — グローバルホットキー（`Ctrl+Shift+Space`）で即座にエディタが開く
- **自動保存** — ウィンドウを閉じると自動的に保存される
- **ライブプレビュー** — Obsidian風のその場で整形表示（左右分割なし）
- **メモ一覧** — サイドバーからメモを一覧・選択できる
- **カスタマイズ可能** — 保存先、フォント、カラーテーマを変更可能（Tokyo Nightがデフォルト）
- **クロスプラットフォーム** — Windows、macOS、Linuxに対応

## インストール

[Releases](https://github.com/your-repo/releases)ページから、お使いのOS用のファイルをダウンロードしてください。

## 使い方

1. `Ctrl+Shift+Space`でエディタを起動
2. Markdownで書く
3. ウィンドウを閉じるか、ホットキーを再度押す — 自動で保存される

### サイドバー

左上のメニューアイコンをクリックすると:
- 保存済みのメモを一覧表示
- 既存のメモを開いて編集
- 設定画面にアクセス

### 設定

- **保存先** — メモを保存するディレクトリ
- **フォント** — フォントファミリー、サイズ、行間
- **テーマ** — Tokyo Night（ダーク）またはLight

## ファイル形式

メモはYAMLフロントマター付きのMarkdownファイルとして保存されます:

```markdown
---
uid: "01J1Z9P6V9WQ7H9QXGQ2K5J1ZC"
created_at: "2025-01-13T06:12:34+09:00"
updated_at: "2025-01-13T06:15:10+09:00"
---

メモの内容...
```

## 開発

```bash
# 依存関係のインストール
bun install

# 開発モードで実行
bun run tauri dev

# 本番用ビルド
bun run tauri build
```

## 技術スタック

- **フロントエンド**: SvelteKit 5 + TypeScript
- **バックエンド**: Rust + Tauri v2
